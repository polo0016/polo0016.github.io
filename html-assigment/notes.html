5 HTML tags

Img Tag
This tag is used to include a picture on an HTML page. It always has to be with the src to specify where the picture is located.
Alt Tag
This tag should always be with the img tag like the src ttribute because in case the image is not loading, it would show in text a description of the picture.
P Tag
This tag is used to create a paragraph inside of a HTML page.
Br tag
This tag is an element that we use to create a line on an HTML page.
HTML Tag
This tag is very important to defined a document as an HTLM document
